# TP4 - K-Means : Segmentation Stratégique (Modèle RFM)

## Description
Pipeline complet de Data Mining pour segmenter les clients 
d'un e-commerce en utilisant le modèle RFM et K-Means Clustering.

## Dataset
Online Retail II UCI - 1,067,371 transactions

## Résultats
4 segments identifiés :
- Champions (1188 clients)
- Clients à Risque (1974 clients)
- Clients Endormis (1465 clients)
- Nouveaux Clients (1251 clients)
